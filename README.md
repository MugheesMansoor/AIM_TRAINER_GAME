# AIM_TRAINER_GAME
Aim Trainer Game built with Python and Pygame.
A fast-paced mouse accuracy game where targets appear randomly, grow and shrink over time, and the player must click them before they disappear. 
The game tracks time, speed, hits, accuracy, and lives, making it a fun way to improve aim and reaction skills.
